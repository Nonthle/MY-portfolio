Cybercrime Dynamics System

SIR Model + Machine Learning for cyber threat propagation

Python TensorFlow Scikit-learn PINNs SIR Model

Problem

Cybercrime incidents are rapidly increasing across digital networks, and traditional security systems struggle to model how attacks spread and evolve over time. Most systems are reactive rather than predictive, making it difficult to understand how vulnerabilities propagate within a network.

Approach

This project applies the SIR (Susceptible–Infected–Recovered) epidemiological model to simulate the spread of cyber threats within a network environment. Users in a system are categorized into three states: Susceptible (S), Infected (I), and Recovered (R).

The model is extended using machine learning: a Decision Tree classifier identifies risk patterns, Physics-Informed Neural Networks (PINNs) model dynamic system behavior, and PCA is applied for dimensionality reduction.

Result

The hybrid model improved prediction accuracy by approximately 30%, demonstrating the effectiveness of combining mathematical epidemiological models with data-driven machine learning for cybersecurity analysis.

Models Used

  • SIR Model
  • Physics-Informed Neural Networks (PINNs)
  • Decision Tree
  • PCA
